# ÀTÚPALẸ̀: Ancient Formula To Unlock Yoruba Words Meaning

Welcome to the official repository for **ÀTÚPALẸ̀**, an interactive linguistic application designed for the Pi Network ecosystem. 

This application serves as a digital companion to the pioneering framework introduced in the ebook *"Àtúpalẹ̀: Ancient Formula To Unlock Yoruba Words Meaning."*

---

## 🌟 Project Overview

**ÀTÚPALẸ̀** is a multifaceted educational tool that breaks down complex, ancient, and modern Yoruba words using a structured linguistic formula. By deconstructing words into their core monosyllables, tonal shifts, and entity markers, the app helps educators, researchers, and language learners uncover the deep semantic roots and etymology embedded within the Yoruba language.

### Key Features:
*   **Syllable Deconstruction (Òrò Ìpìlẹ̀):** Break down words to understand how individual syllables build meaning.
*   **Entity Marker Analysis (Àfikún):** Explore how specific vowel prefixes (A, E, Ẹ, I) serve as structural foundations for naming individuals, places, or items.
*   **The Word Lab (Àtúpalẹ̀):** An interactive interface where users can deconstruct terms dynamically.
*   **Cultural Tonal Studies:** Learn how shifts in accent and tone completely transition the definition and spiritual context of a word.
